# Predictive Maintenance Analysis for Aircraft Engines

## Overview

This project applies machine learning and statistical analysis techniques to predict aircraft engine failures and estimate Remaining Useful Life (RUL). The objective is to support predictive maintenance strategies by identifying potential failures before they occur, enabling proactive maintenance planning and reducing operational downtime.

The analysis uses sensor measurements collected from aircraft engines operating under simulated run-to-failure conditions. Multiple regression and classification algorithms are implemented to analyze engine health and predict future failures.

---

## Problem Statement

Aircraft engine failures can result in costly maintenance operations, unexpected downtime, and reduced operational efficiency. Traditional maintenance approaches often rely on fixed schedules or reactive repairs.

This project explores how machine learning models can be used to:

* Predict engine degradation trends
* Estimate Remaining Useful Life (RUL)
* Identify engines at risk of failure
* Support data-driven maintenance decisions

## Machine Learning Models

### Regression Models

#### Elastic Net Regression

Used to estimate Remaining Useful Life (RUL) by combining L1 and L2 regularization techniques.

#### Random Forest Regressor

Used to capture complex non-linear relationships within engine sensor data and improve prediction accuracy.

### Classification Models

#### K-Nearest Neighbors (KNN)

Used to classify engine health conditions and identify engines likely to fail during operation.

## Dataset

The dataset used in this project is based on a simulated aircraft engine run-to-failure scenario.

### Files

#### train_selected.csv

Contains sensor measurements from 100 aircraft engines operating until failure.

* Shape: (20631, 8)

#### test_selected_ttf.csv

Contains sensor measurements used for evaluating failure prediction and RUL estimation.

* Shape: (100, 8)

The degradation pattern of each engine is reflected through multiple sensor measurements collected during operation.
